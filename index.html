Coding Flash Cards

Javascript (JS)

JavaScript: Object-oriented computer programming language commonly used to create interactive effects within web browsers

Operators

Operators: An operator is a symbol that is used to perform an operation. Most often these operations are arithmetic (addition, subtraction, etc), but not always

Variables

Variables: containers for storing data values

Variable Naming Conventions

Variable Naming Conventions: JavaScript variable names should not start with a numeral (0-9). They must begin with a letter or an underscore character. JavaScript variable names are case-sensitive, also use camelCase

Functions

Functions: A set of statements that performs a task or calculates a value

Methods

Methods: Actions that can be performed on objects

Arguments

Arguments: Parameters are variables listed as a part of the function definition. Arguments are values passed to the function when it is invoked

Parameters

Parameters: Parameters are variables listed as a part of the function definition. Arguments are values passed to the function when it is invoked

Return

Return: Functions always return a value. In JavaScript, if no return value is specified, the function will return undefined

Chaining Methods

Chaining Methods: Also known as Cascading, refers to repeatedly calling one method after another on an object, in one continuous line of code

Strings

Strings: Are used for storing and manipulating text and is zero or more characters written inside quotes

Booleans

Booleans: A datatype that returns either of two values i.e. true or false

Undefined

Undefined: A primitive value used when a variable has not been assigned a value

NaN

NaN: This property represents "Not-a-Number" value, a part of the numbers datatype. This property indicates that a value is not a legal number

Comments

Comments: Denoted with a "//", comments allow you to make comments in code without it appearing in the code

JQuery

JQuery: A concise and fast JavaScript library that can be used to simplify event handling, HTML document traversing, Ajax interactions and animation for speedy website development
